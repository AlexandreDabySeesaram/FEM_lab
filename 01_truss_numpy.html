1D Linear Elasticity FEM (NumPy)

1D uniaxial bar under traction and body forces

Formulation of the problem

Strong form of the problem

For a 1D beam under tension-compression, if an external distributed axial force \(q(x)\) is applied per unit length, the conservation of linear momentum reads

\[\frac{\mathrm{d}N}{\mathrm{d}x} + q = 0,\]

where \(N(x) = \int_A \sigma \, \mathrm{d}A = \sigma A\) represents the resultant axial force. The kinematics is described using the linearised strain tensor

\[\varepsilon = \frac{\mathrm{d}u}{\mathrm{d}x} = u'(x),\]

which is proportional to Cauchy stress tensor in linear elasticity, therefore

\[\sigma = E\varepsilon\], with \(E\) the young modulus of our beam.

Therefore,

\[\frac{\mathrm{d}}{\mathrm{d}x}\left(EA \frac{\mathrm{d}u}{\mathrm{d}x}\right) + q = 0\]

Assuming homogeneous material properties and a constant cross-section (\(EA = \text{constant}\)), we arrive at your governing beam equation under tension:

\[E A u''(x) + q = 0\]

Weak forumlation

We multiply the strong form by a virtual displacement test function \(v(x)\in \mathcal{H^p_0\left(\Omega\right)}\) and integrate over the domain \([0, L]\):

\[\int_0^L v(x) \left[ EA u''(x) + q \right] \mathrm{d}x = 0\]

Applying integration by parts to the second-derivative term:

\[\left[ v(x) EA u'(x) \right]_0^L - \int_0^L v'(x) EA u'(x) \mathrm{d}x + \int_0^L v(x) q \mathrm{d}x = 0\]

Because \(v(x)\) is homegeneous, the weak form reads

\[\int_0^L EA u'(x) v'(x) \mathrm{d}x = \int_0^L q v(x) \mathrm{d}x\]

Boundary conditions

We apply Dirichlet boundary conditions at both ends of the beam

  • \(u(x = 0) = u_0\)
  • \(u(x = L) = u_L\)

Discretisation

We discretise the 1D bar of length \(L\) with \(N_e\) elements.

We compare linear (\(P^1\)) and quadratic (\(P^2\)) elements, use different Dirichlet boundary conditions formulations, and analyse convergence rates.

Exact Analytical Solution

By integrating \(E A u''(x) + q = 0\) twice and applying boundary conditions, we get: \[u_{exact}(x) = -\frac{q}{2EA}x^2 + C_1 x + u_0\] where \(C_1 = \frac{u_L - u_0}{L} + \frac{q L}{2EA}\)

The corresponding strain is: \[\varepsilon_{exact}(x) = u'_{exact}(x) = -\frac{q}{EA}x + C_1\]

Code 
def u_exact(x):
    """Exact analytical displacement."""
    C1 = (uL - u0) / L + q * L / (2.0 * E * A)
    return -q / (2.0 * E * A) * x**2 + C1 * x + u0

def eps_exact(x):
    """Exact analytical strain."""
    C1 = (uL - u0) / L + q * L / (2.0 * E * A)
    return -q / (E * A) * x + C1

Solving the problem with different methods

We solve the problem on a grid with \(N_e = 4\) elements using:

  1. P1 (Linear) elements with penalisation
  2. P1 (Linear) elements with Lagrange Multipliers
  3. P2 (Quadratic) elements with penalisation
  4. P2 (Quadratic) elements with Lagrange Multipliers

Mesh Convergence Analysis

We run the P1 and P2 element solvers on a sequence of refined meshes: \[N_e \in \{4, 8, 16, 32, 64\}\] We compute the \(L^2\) error norm for displacements using a 3-point Gauss quadrature: \[\text{Error}_{L^2} = \sqrt{\int_0^L (u_{exact}(x) - u_h(x))^2 \, \mathrm{d}x}\]

Different loading conditions are studied.

Constant Body Force

Solve

from fem_lib import gauss_quadrature, shape_functions

def compute_l2_error(x_nodes, elements, element_type, u_numeric, u_exact_fn):
    """Computes the exact L2 error norm using Gauss-Legendre quadrature."""
    error_sq = 0.0
    xi_g, w_g = gauss_quadrature(3)
    
    for elem in elements:
        x_el = x_nodes[elem]
        x_start = x_el[0]
        x_end = x_el[-1]
        Le = x_end - x_start
        J = Le / 2.0
        u_el = u_numeric[elem]
        
        for xi, w in zip(xi_g, w_g):
            N, _ = shape_functions(element_type, xi)
            u_h_val = np.dot(N, u_el)
            
            x_val = 0.5 * (x_start + x_end) + J * xi
            u_ex_val = u_exact_fn(x_val)
            
            error_sq += (u_ex_val - u_h_val)**2 * J * w
            
    return np.sqrt(error_sq)

def compute_energy_error(x_nodes, elements, element_type, u_numeric, E_val, A_val, eps_exact_fn):
    """Computes the exact energy error norm using Gauss-Legendre quadrature."""
    error_sq = 0.0
    xi_g, w_g = gauss_quadrature(3)
    
    for elem in elements:
        x_el = x_nodes[elem]
        x_start = x_el[0]
        x_end = x_el[-1]
        Le = x_end - x_start
        J = Le / 2.0
        u_el = u_numeric[elem]
        
        for xi, w in zip(xi_g, w_g):
            _, dN_dxi = shape_functions(element_type, xi)
            duh_dx = np.dot(dN_dxi, u_el) / J
            
            x_val = 0.5 * (x_start + x_end) + J * xi
            E_p = E_val(x_val) if callable(E_val) else E_val
            A_p = A_val(x_val) if callable(A_val) else A_val
            
            duex_dx = eps_exact_fn(x_val)
            
            error_sq += E_p * A_p * (duex_dx - duh_dx)**2 * J * w
            
    return np.sqrt(error_sq)

# Run convergence study for constant body force
N_list = [4, 8, 16, 32, 64]
h_list = []
errors_p1 = []
errors_p2 = []
energy_errors_p1 = []
energy_errors_p2 = []

    h = L / Ne_loop
    h_list.append(h)
    
    # Solve P1
    nodes_p1_loop = Ne_loop + 1
    x_p1_loop = np.linspace(0.0, L, nodes_p1_loop)
    elements_p1_loop = np.array([[j, j+1] for j in range(Ne_loop)])
    K_p1_l, F_p1_l = assemble_global_system(x_p1_loop, elements_p1_loop, "P1", E, A, q)
    fixed_dofs_p1_l = [0, nodes_p1_loop - 1]
    fixed_vals_p1_l = [u0, uL]
    K_p1_lag_l, F_p1_lag_l = apply_bc_lagrange(K_p1_l, F_p1_l, fixed_dofs_p1_l, fixed_vals_p1_l)
    sol_p1 = np.linalg.solve(K_p1_lag_l, F_p1_lag_l)
    u_p1_sol = sol_p1[:nodes_p1_loop]

    # Solve P2

Remark: Polynomial Reproduction & Machine Precision

  1. P1 (Linear) elements show clear convergence rates:
    • L2 Displacement Error: Converges at \(\mathcal{O}(h^2)\) as expected.
    • Energy Error: Converges at \(\mathcal{O}(h^1)\) (matches the derivative/strain accuracy).
  2. P2 (Quadratic) elements do not show convergence trends because the exact displacement solution is a quadratic polynomial, which \(P^2\) shape functions span exactly. Thus, both the displacement (\(P^2\)) and its derivative (\(P^1\) strain) are reproduced exactly up to double-precision machine limits (\(\approx 10^{-16}\)).

Sinusoidal Body Force

To demonstrate the theoretical convergence rates when the exact solution is not a low-order polynomial, we now apply a sinusoidal body force: \[q(x) = q_0 \sin\left(\frac{\pi x}{L}\right)\]

Integrating \(E A u''(x) + q(x) = 0\) twice with boundary conditions \(u(0) = u_0\) and \(u(L) = u_L\) yields the exact solution: \[u_{\text{exact}}(x) = \frac{q_0 L^2}{\pi^2 E A} \sin\left(\frac{\pi x}{L}\right) + \frac{u_L - u_0}{L}x + u_0\]

Solve

q0 = 2e5

def u_exact_sin(x):
    """Exact analytical displacement for sinusoidal body force."""
    return (q0 * L**2) / (np.pi**2 * E * A) * np.sin(np.pi * x / L) + ((uL - u0) / L) * x + u0

def eps_exact_sin(x):
    """Exact analytical strain for sinusoidal body force."""
    return (q0 * L) / (np.pi * E * A) * np.cos(np.pi * x / L) + (uL - u0) / L

def q_sin_func(x):
    """Sinusoidal body force function."""
    return q0 * np.sin(np.pi * x / L)

    u_p2_sol = sol_p2[:nodes_p2_loop]
    
    err_p2 = compute_l2_error(x_p2_loop, elements_p2_loop, "P2", u_p2_sol, u_exact_sin)
    errors_p2_sin.append(err_p2)
    
    en_err_p2 = compute_energy_error(x_p2_loop, elements_p2_loop, "P2", u_p2_sol, E, A, eps_exact_sin)
    energy_errors_p2_sin.append(en_err_p2)
